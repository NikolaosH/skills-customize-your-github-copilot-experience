# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Master the use of Python's built-in data structures: lists, dictionaries, sets, and tuples. Learn to create, manipulate, and apply these structures in various programming scenarios.

## 📝 Tasks

### 🛠️ Working with Lists

#### Description
Create and manipulate lists to store and organize data. Practice adding, removing, and accessing list elements.

#### Requirements
Completed program should:

- Create lists with different data types
- Use indexing and slicing to access elements
- Add and remove elements using append, insert, remove, pop
- Sort and reverse lists
- Iterate through lists using loops

### 🛠️ Using Dictionaries

#### Description
Implement dictionaries to store key-value pairs. Practice accessing, updating, and iterating over dictionary data.

#### Requirements
Completed program should:

- Create dictionaries with various key-value pairs
- Access values using keys
- Add, update, and delete key-value pairs
- Check for key existence
- Iterate through keys, values, or items

### 🛠️ Exploring Sets and Tuples

#### Description
Work with sets for unique collections and tuples for immutable sequences. Understand their differences and use cases.

#### Requirements
Completed program should:

- Create sets and perform set operations (union, intersection, difference)
- Create tuples and demonstrate their immutability
- Convert between lists, sets, and tuples
- Use sets to remove duplicates from lists
- Compare performance and use cases of each data structure
